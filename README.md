# TicTacToe

This is an interactive Tic-Tac-Toe game where you play against an **unbeatable AI** powered by the **Minimax algorithm with Alpha-Beta Pruning**. 
The Goal: To demonstrate **Adversarial Search algorithms** in a graphical environment using Pygame.

## Algorithm Used

### Minimax with Alpha-Beta Pruning

- **Minimax** simulates all possible game outcomes
- AI assumes:
  - It plays optimally
  - Opponent also plays optimally
- **Alpha-Beta Pruning** improves efficiency by:
  - Skipping unnecessary branches
  - Reducing computation time
## The Tech Stack
Language: Python 3.x
Library: Pygame (for GUI, event handling, and rendering).
Algorithm: Minimax with Alpha-Beta Pruning.

## Install Dependencies
Ensure you have pygame installed:

```bash
pip install pygame
```

## Run the Game
Execute the main script to start the application:
```bash
python main.py
```

### Controls
 - Mouse Left-Click: Place your mark (X) on the grid.
 - 'R' Key: Restart the game.
 - 'ESC' or Close Window: Exit the application.
